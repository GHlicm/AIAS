# 目标检测(支持coco数据集分类)SDK
检测图片中80个分类的目标。

### 支持分类如下：
- person
- bicycle
- car
- motorcycle
- airplane
- bus
- train
- truck
- boat
- traffic light
- fire hydrant
- stop sign
  ...

[点击下载](https://aias-home.oss-cn-beijing.aliyuncs.com/AIAS/object_detection_sdk/coco_classes.txt)
